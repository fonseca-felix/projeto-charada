# Desafio do Charada ❓🟩

Bem-vindo ao **Desafio do Charada**, um jogo interativo construído em JavaScript focado em enigmas onde seu raciocínio lógico é a única salvação para escapar das armadilhas da sua mente. Prepare-se para decifrar o que é indecifrável em uma interface sombria, misteriosa e em tons de verde neon.

## 🔗 Links da Infraestrutura

Para consumir as palavras e os enigmas, este frontend consome uma infraestrutura exclusiva:

- **API Oficial das Charadas:** [https://charada-orpin.vercel.app](https://charada-orpin.vercel.app)
- **Repositório GitHub do Backend:** [https://github.com/fonseca-felix/charada.git](https://github.com/fonseca-felix/charada.git)

---

## 🎯 Modos de Jogo

- **Modo Investigação (Casual)**: Para mentes cautelosas. O tempo não existe aqui. Concentre-se apenas na sabedoria.
- **Modo Sob Pressão (Cronômetro 30s)**: O relógio avança. Descubra a resposta em até 30 segundos ou receba uma penalidade no seu histórico de acertos.

## 🛠️ Tecnologias Utilizadas (Frontend)

- **HTML5 & CSS3 / TailwindCSS**
- **JavaScript Vanilla Moderno** (Engine baseada no fluxo flexível do DOM e uso extensivo de classes para estado)
- Integração de UI via `fetch` assíncrono

## ⚙️ Características Premium

1. **Validação Inteligente** (`script.js`):
   O motor JavaScript foi arquitetado para ser tolerante. O jogador **não** perderá se esquecer de colocar um acento ou letra maiúscula. A API de validação remove pontuações, minúsculas e captura palavras-chave, gerando zero estresse no gameplay.
   
2. **Interface Gotham Edge** (`index.html`):
   O layout foi montado utilizando a estética clássica preta e verde do temível Charada, com micro-interações incluídas:
   - *Shake* contundente da tela com borda vermelha quando você falha.
   - *Glow* Verde Neon de comemoração sob respostas geniais.
